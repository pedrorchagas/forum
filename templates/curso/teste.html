Tipos de marcação de texto

Negrito

Usa a tag "strong" para deixar os textos negritos

Sublinhadas

Usa a tag "em" para deixar os textos sublinhados

Marca texto

usa a tag "mark" para marcar os textos

Textos deletados

usa a tag "del" para deixar os textos deletados (dar enfâse que o texto foi apagado mas deve ser lido)

Textos inseridos

usa a tag "ins" para deixar os textos incluidos (dar enfâse que o texto foi incluido)

Elevar textos e rebaixar

usa a tag "sup" para elevar textos e

a tag "sub" para rebaixar textos

Tag code

usa a tag para escrever códigos

Tag pre-formatado

usa a tag 

            para escrever 
            i 
            gual
            ao código fonte

Citação

Usa a tag "q" para dizer que é uma citação

ou "blockquote" para citações maiores

Abreviações

usa a tag "abbr" para abreviações como: PY
